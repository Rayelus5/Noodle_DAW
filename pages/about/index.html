Misión del portal web

Este portal web nace con el objetivo de acercar el trabajo del alumnado con el profesorado de los módulos impartidos en el Ciclo Formativo de Grado Superior de Desarrollo de Aplicaciones Web (DAW). El código de esta web tanto como el de las actividades es OpenSource, es decir, público y de código abierto. Puede ser compartida con fines educativos.

Este portal está pensado para ser utilizado para el alumnado que curse dicho curso, y puedan acceder a toda la información que necesiten para realizar sus tareas o llevar todo al día.
